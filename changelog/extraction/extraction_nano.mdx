title: "Extraction Nano"
description: "Changelog for the Extraction Nano base processor in Extend."

Our most cost-effective base extraction processor. Optimized for low latency, extremely high volume use cases where cost at scale is a key consideration. Not suitable for more complex tasks like large table extraction or signature detection.
